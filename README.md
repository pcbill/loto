

**抽獎系統 (Loto)**，從專案結構來看，它是一個基於 Node.js 的 Web 應用程式。 
根據內容，這是一個**屏東科技大學 (NPUST) 年終尾牙抽獎系統**。

## 主要功能

### 1. 📝 登錄系統
- 輸入員工編號，登錄進系統參加抽獎
- 顯示已登錄訊息

### 2. 👥 人員管理
- 顯示登錄總人數
- 登錄列表（員編號、姓名、登錄時間、中獎獎項、中獎時間、兌獎時間）
- 依員編或姓名搜尋個人

### 3. 🎁 獎項管理
- 顯示所有抽獎資訊
- 查詢單一抽獎詳細資訊
- 增加抽獎（可選全員或目前未中獎者）

### 4. 🎰 執行抽獎
- **一次抽多個**：依照獎品數量，從未中獎列表隨機抽出相對數量
- **一次抽一個**：戲劇性的表現效果（有視覺動畫）

### 5. ✅ 兌獎功能
- 輸入員編確認中獎
- 登記已兌獎狀態

## 技術架構
- **後端**: Node.js + Express 4
- **前端**: EJS 模板引擎
- **資料庫**: PostgreSQL
- **特效**: jQuery jSlots (拉霸機動畫效果)

這是一個完整的尾牙/年會抽獎管理系統！
